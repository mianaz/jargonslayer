为什么WHY

非英语母语者卡住的从来不是单词

Non-native speakers rarely get stuck on vocabulary

01
Non-literal expressions
卡在 非字面表达 —— move the needle、boil the ocean、table this。每个词都认识，句子却拼不出意思；等反应过来，会议已经翻篇。 You know every word in move the needle, boil the ocean, table this — the sentence still doesn't parse, and by the time it does, the meeting has moved on.
02
Proper nouns & acronyms
卡在 专有名词与缩写 —— ARR、OKR、Series B、内部代号。母语者一秒扫过去，你花那一秒去检索。 ARR, OKR, Series B, internal codenames. A native speaker skips past them in a second; you spend that second retrieving.

JargonSlayer 是坐在你旁边的同事：它从不打断，只是在侧边栏告诉你那句话「实际」是什么意思。 JargonSlayer is the colleague sitting next to you: it never interrupts, it just tells you in the sidebar what that sentence actually meant.

功能FEATURES

功能一览

Features

real-time transcription
实时转录：浏览器识别（云端）/ 本地 Whisper / 标签页音频（后两者音频不出本机），每个引擎带「本地 / 云端」数据去向标识；内置演示零依赖看完整流程。 Browser speech recognition (cloud), local Whisper, or tab audio (the latter two never leave the machine). Every engine is labeled local / cloud so the data path is visible at a glance; a zero-dependency demo shows the whole flow without a microphone or key.
bilingual transcript
双语字幕：开启后英文每出一句，中文译文即刻跟在下方——会议还在开，你已经读完了；不用等会后报告再回头猜刚才讲了什么。 Live bilingual subtitles: each English sentence gets a Chinese translation the moment it lands — read along while the meeting is still running instead of deciphering a post-meeting summary after the fact.
import anything
万物导入：粘贴文稿（.txt / .srt / .vtt）直接解析；丢音频或视频文件进来——Whisper 在浏览器内转录（WebGPU 可用走 WebGPU，否则 WASM；视频经 ffmpeg.wasm 自动剥音轨），文件全程不上传；体验版开箱即用，装了本地 sidecar 还能直接贴视频链接拉取。 Import anything: paste a transcript (.txt / .srt / .vtt) for instant parsing, or drop in an audio or video file — Whisper transcribes right in the browser (WebGPU when available, WASM otherwise; video tracks are extracted via ffmpeg.wasm automatically). Nothing gets uploaded. The hosted demo works out of the box; sidecar users can also import directly from a video URL.
expression detection
实时表达检测：LLM 结合前后语境判断，只解释"字面意思 ≠ 实际意思"的表达，专有名词/缩写单独成术语条。没有 API Key 时自动用内置词典（370+ 条、10 个主题包），还能从 GitHub 安装社区词典包。解释语言可切中文或英文。 Real-time expression detection: an LLM uses surrounding context to explain only what "literal ≠ actual" (it can tell whether table this means shelving a topic or furniture); proper nouns get separate term cards. Without an API key it falls back to a built-in dictionary (370+ entries, 10 topic packs) and can install community packs from GitHub.
card experience
卡片体验：表达/术语共享同一块状样式，分类色左边条区分；重复出现只计数不刷屏；转录内高亮表达可点定位；选中任意文字即席查询并可一键收进「我的词典」。 Card experience: expression and term cards share one block style with category-colored status bars; repeats increment a counter instead of flooding the feed; underlined expressions in the transcript jump to their card; select any text for an ad-hoc lookup and one-click save to your glossary.
speaker diarization
说话人：上传录音自动转录 + 说话人分离（后台进度，完成自动载入）；实时分离 beta（本地运行，标签随会议逐步修正）；说话人标签点击即可改名。 Speakers: import a recording for offline transcription + speaker diarization (background job, auto-loads when done), or turn on realtime diarization (beta, local, labels refine as the meeting proceeds). Click a speaker label to rename it.
post-meeting artifacts
会后产物：双语纪要（主题/要点/决定/行动项）、逐段对照全文、学习卡片、康奈尔笔记（正文高亮 + 右栏批注 + 小结，可导出 PNG/Markdown）；另有 Markdown / Anki TSV / JSON 导出与自动落盘、webhook。 Post-meeting artifacts: bilingual summary (topic / key points / decisions / action items), paragraph-aligned translation, study cards, and a Cornell-note sheet (highlighted body + margin annotations, exportable as PNG or Markdown); plus Markdown / Anki TSV / JSON export, auto-save to a folder, and a webhook.
learning center
学习中心：/review 页有统计、高频 Top 10、词云、翻卡练习；个人词典参与后续会议的检测。 Learning center: /review has stats, a frequency Top 10, a word cloud, and a flashcard practice deck; your personal glossary feeds back into detection in later meetings.
BYOK, multi-model
BYOK / 多模型：Anthropic 直连或任意 OpenAI 兼容端点（DeepSeek / Qwen / OpenRouter / Ollama），Key 只存本机浏览器；OpenRouter 支持一键 OAuth 登录自动生成 Key，Poe 订阅用户有内置预设，接入零配置。 BYOK, multi-model: Anthropic direct or any OpenAI-compatible endpoint (DeepSeek / Qwen / OpenRouter / Ollama). The key lives only in your local browser. OpenRouter users can connect with one-click OAuth — the key is generated automatically; Poe subscribers get a built-in preset with zero configuration.
history, no accounts
会议历史：全部存浏览器 IndexedDB，无账号，支持搜索曾出现过的表达；一键全量备份/恢复。 History without accounts: everything in IndexedDB; search old meetings by expression; one-click full backup and restore.

快速开始QUICKSTART

快速开始

Quickstart

bash
# 克隆仓库clone the repo
git clone https://github.com/mianaz/jargonslayer.git
cd jargonslayer
npm install
npm run dev
# 打开open http://localhost:3000

第一次打开会弹出新手引导。先点右上角「演示」 —— 不需要麦克风、不需要 API Key，就能看到完整的转录 → 检测 → 卡片 → 会后报告流程（无 Key 时演示走内置词典）。 On first launch an onboarding tour appears. Open the ≡ menu and click 「演示」 (Demo) first — no microphone, no API key, and you'll see the full transcription → detection → cards → report flow (the demo runs on the built-in dictionary when no key is configured).

状态栏上蹲着的像素龙叫 Bit：像光标一样眨眼的方块瞳孔，监听时背鳍逐格点亮如信号计，新卡片落地时喷出 ANSI 彩色像素火焰。会议结束 30 秒无动作它就会睡着。 The pixel dragon perched on the status line is Bit — cursor-block pupils that blink like a caret, dorsal fins that light up like a signal meter while listening, and ANSI-colored pixel fire when a new card lands. It sleeps 30 seconds after a meeting ends.
